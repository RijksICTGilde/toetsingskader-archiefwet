{{- /* Render een link met automatische detectie van externe links. Parameters (dict): - href: De URL - text: De linktekst - title: (optioneel) Title attribuut - private: (optioneel) Markeer als besloten link via {.private} in Markdown Externe links krijgen automatisch target="_blank" en rel="external noopener noreferrer". Private links krijgen een slotje-icoon en "(besloten omgeving)" label. Een link is private als {.private} is meegegeven in Markdown, of automatisch bij samenwerkruimten.nl en digilab.overheid.nl/chat. */ -}} {{- $u := urls.Parse .href -}} {{- $isExternal := and $u.IsAbs (hasPrefix $u.Scheme "http") -}} {{- $isPrivate := or .private (and $isExternal (or (strings.Contains $u.Host "samenwerkruimten.nl") (and (strings.Contains $u.Host "digilab.overheid.nl") (strings.Contains $u.Path "/chat")))) -}} {{- $class := .class -}} {{- if $isPrivate -}} {{ .text }} (besloten omgeving) {{- else if $isExternal -}} {{ .text }} {{- else -}} {{ .text }} {{- end -}}
